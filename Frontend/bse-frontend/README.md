# BSE Frontend

Simple React + Vite frontend shell.

This project currently has no page routing and no backend API wiring. It is only the starting UI shell.

## Run

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Current Structure

```text
src/App.jsx
src/main.jsx
index.html
vite.config.js
```

## Current Rule

Keep this frontend disconnected until one backend contract is chosen and approved. Add only one connection at a time.
